## Cómo Ejecutar

Primero hay que pararse dentro de la carpeta [`run`](run), o de la carpeta raíz, y ejecutar uno de los dos
archivos presentes en [`run`](run), según el Sistema Operativo.

### **Ejemplos:**
```bat
@REM para Windows
cd run
./run.bat
```
```sh
# para Linux u otro entorno de shell
cd run
./run.sh
```
